================================================================================
IMPORTANT NOTES
================================================================================

 Security for Production:
   - Change secret key in app.py
   - Use environment variables
   - Add password hashing
   - Use PostgreSQL instead of SQLite

 Database on Deployment:
   - SQLite file won't be pushed (in .gitignore)
   - New empty database created on deployment
   - Admin account auto-created (admin/admin123)
   - You'll need to add books again OR modify app.py to auto-populate

 Making Updates:
   git add .
   git commit -m "Your update message"
   git push origin main
   (Deployment platforms auto-deploy)
